Overview
Welcome to our Insurance Charge Prediction website! This project shows the power of machine learning to predict insurance charges. Built using Python, Flask, HTML and CSS, our platform provides a user-friendly interface for accurate predictions.

Technology used
HTML & CSS : The frontend is built using HTML and CSS, ensuring a visually appealing design.
Python & Flask : The backend is powered by Python and Flask.
Machine Learning : Our prediction model is based on a Random Forest algorithm.

How it works
Input Data : Users enter their personal details such as age, gender, BMI, number of children, smoking status, and region.
Prediction : The data is processed by our machine learning model, which predicts the insurance charges.
Result : The predicted insurance charges are displayed instantly, helping users understand their insurance costs.
